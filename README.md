PriceFinderPro-NoLogin: Product Search & Price Comparison Tool

Introduction

PriceFinderPro-NoLogin is a standalone desktop application designed to help users search for products across multiple Vietnamese e-commerce websites, compare prices, and save favorite product links. Unlike other platforms, this application does not require login or registration, making it quick and convenient for immediate use.

The program is built using Java Swing for the user interface, providing an intuitive and responsive GUI. It simulates product search results and pricing for popular websites including Shopee, Lazada, Tiki, Sendo, Thế Giới Di Động, Điện máy XANH, FPT Shop, CellphoneS, Hoàng Hà Mobile, Phong Vũ, Nguyễn Kim, An Phát, GearVN, Hanoicomputer, Meta.vn, and more.

With PriceFinderPro-NoLogin, users can:

Search for products using keywords (supports Vietnamese characters).

Filter results by specific websites.

Compare prices among different sellers.

Save favorite product links for future reference.

View keyword suggestions based on history and Google Suggest.

Navigate results with pagination.

Analyze price ranges and find the cheapest or most expensive options.

This makes it an ideal tool for online shoppers, price-conscious consumers, or anyone who wants to quickly gather product pricing information across multiple sellers.

Features Overview

Keyword Search

Enter a product name or keyword into the search bar.

Supports Vietnamese and accented characters.

Press Enter or click Search to generate results.

Search Suggestions

As you type, the application shows suggestions based on your previous searches and Google Suggestions.

Navigate suggestions using arrow keys and select with Enter or double-click.

Result Display

Search results are displayed in a list showing the product title, website, and URL.

Each page shows up to 20 results.

Use Previous / Next buttons to navigate between pages.

Website Filter

On the left panel, you can select one or multiple websites to filter results.

Only results from selected websites will be shown.

Price Comparison

Click Compare Products to open a comparison dialog.

Filter products by price range using Minimum and Maximum price selectors.

Quickly find the lowest or highest priced product.

Compare two selected websites to see which one offers the product cheaper and the price difference.

Saving Favorite Products

Right-click a search result and choose Save this website to store it in the bookmarks list.

View saved products by clicking Saved Websites.

Double-click a saved item to open it in your web browser.

Delete saved items individually if no longer needed.

Keyword History Management

Automatically saves your searched keywords.

Provides auto-suggestions for faster searches.

Click Clear Keyword History to remove all saved search terms.

Pagination

Results are paginated, with controls to move to the previous or next page.

Displays current page number and total results.

Simulated Prices

Prices are generated using a deterministic simulation for demonstration purposes.

Price comparisons reflect these simulated values.

How to Use PriceFinderPro-NoLogin

Launching the Application

Run the program using Java (java -jar PriceFinderPro_NoLogin.jar) or from your IDE.

The main window will appear with a search bar, website filter panel, and results panel.

Searching for a Product

Type the product keyword in the search bar.

Press Enter or click Search.

Results will populate the right panel.

Filtering by Website

Select one or multiple websites from the left panel to see results only from those sites.

The results list updates automatically when you change the selection.

Saving Products

Right-click on a search result and select Save this website.

View your saved products by clicking Saved Websites.

Manage saved products with delete or double-click to open in browser.

Comparing Prices

Click Compare Products to open the comparison dialog.

Use Min/Max Price filters to narrow down products.

Click Lowest Price or Highest Price for quick analysis.

Select two websites from the right panel and click Compare 2 Websites to see which is cheaper.

Keyword Suggestions

As you type in the search bar, suggestions will appear.

Use arrow keys to select or double-click to choose a suggestion.

Clearing Keyword History

Click Clear Keyword History to remove all saved keywords.

This action is permanent.

System Requirements

Java Runtime Environment (JRE) 8 or higher.

Internet connection (for Google suggestions).

Desktop environment with browser support (to open product URLs).

Notes

The program simulates prices, so actual product prices may differ when visiting the websites.

Saved products and keyword history are stored locally in the user's home directory.

All UI text supports Vietnamese characters.
